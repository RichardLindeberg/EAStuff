# Transaction Monitoring Calibration Instruction (EU AMLR)

**Purpose:**
Define how monitoring scenarios, thresholds, and typologies are calibrated to meet EU AMLR expectations.

## Scope
- Transaction monitoring rules, scenarios, and alert thresholds.
- All products and customer segments.

## Prerequisites
- Monitoring performance metrics
- Alert case quality review results
- Current typology library

## Procedure
1. Review alert quality metrics and false-positive rates.
2. Assess typology coverage against current EU AMLR risks.
3. Propose threshold changes and scenario updates.
4. Validate changes with historical data and independent review.
5. Obtain governance approval before deployment.
6. Document tuning decisions and post-implementation results.

## Outputs and Records
- Calibration decision record
- Approved rule and threshold changes
