# Model Validation and Monitoring Instruction


**Purpose:**
Provide steps for independent validation, approval, and ongoing monitoring of models.

## Scope
- Material models used in credit, fraud, pricing, and risk decisions.

## Prerequisites
- Model documentation package
- Approved model inventory entry

## Procedure
1. Confirm model classification and risk tier.
2. Review data lineage, assumptions, and limitations.
3. Execute validation tests and benchmarking.
4. Document validation findings and remediation actions.
5. Approve model for use or require rework.
6. Define monitoring thresholds and review cadence.

## Outputs and Records
- Validation report and approval record
- Monitoring plan and thresholds
