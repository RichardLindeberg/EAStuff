# Data Subject Access Request (DSAR) Instruction


**Purpose:**
Provide a step-by-step process for handling data subject access requests in line with GDPR requirements.

## Scope
- Customer and employee personal data requests.
- Requests received via online, email, or postal channels.

## Prerequisites
- Verified identity of the requester
- DSAR intake record and tracking ID

## Procedure
1. Log the request and confirm identity within required timelines.
2. Identify all systems and records containing the subject's data.
3. Collect data extracts and review for exemptions.
4. Obtain privacy review and approval for release.
5. Deliver the response securely and update the request record.

## Outputs and Records
- DSAR response package
- Audit trail of review and delivery
